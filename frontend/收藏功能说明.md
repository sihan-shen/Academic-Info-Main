# 收藏功能使用说明

## 功能概述
用户可以在合作项目详情页收藏感兴趣的项目，收藏的项目将显示在首页"我的收藏"中的"收藏项目"标签页。

## 主要功能

### 1. 收藏项目
- 在合作项目详情页面点击"收藏项目"按钮
- 成功后按钮文字变为"已收藏"，背景色变为浅蓝色
- 显示"收藏成功"提示

### 2. 取消收藏
- 在详情页点击"已收藏"按钮可取消收藏
- 显示"已取消收藏"提示
- 按钮恢复为"收藏项目"状态

### 3. 查看收藏
- 从首页点击"我的收藏"图标
- 切换到"收藏项目"标签页
- 显示所有已收藏的项目列表

### 4. 项目卡片信息
收藏列表中每个项目显示：
- 项目标题
- 项目标签（产学研合作、AI技术应用等）
- 负责人/合作方
- 项目金额
- 项目进度（基于成功率）
- 收藏日期

### 5. 跳转详情
- 点击收藏列表中的任意项目卡片
- 跳转到该项目的详情页
- 可以再次查看完整信息

## 技术实现

### 数据存储
- 使用微信小程序本地存储（wx.Storage）
- 存储键名：`favorite_projects`
- 数据格式：JSON数组

### 核心文件
1. `utils/storage.js` - 收藏管理工具类
2. `pages/coop-detail/` - 项目详情页（收藏功能）
3. `pages/favorites/` - 收藏列表页（显示收藏）

### 数据结构
```javascript
{
  id: Number,              // 项目ID
  title: String,           // 项目标题
  tags: Array,             // 标签数组
  partner: String,         // 合作方
  startDate: String,       // 开始日期
  budget: String,          // 项目金额
  description: String,     // 项目描述
  successRate: String,     // 成功率
  favoriteDate: String,    // 收藏时间（ISO格式）
  favoriteTimestamp: Number // 收藏时间戳
}
```

## 使用流程

1. **浏览项目**
   - 首页 → 查合作 → 学术合作挖掘
   - 或：首页 → 查合作 → 学术合作挖掘 → 潜在合作项目

2. **查看详情**
   - 点击任一项目的"查看合作详情"按钮
   - 进入项目详情页

3. **收藏项目**
   - 在详情页底部点击"收藏项目"
   - 查看成功提示

4. **查看收藏**
   - 返回首页
   - 点击"我的收藏"图标
   - 切换到"收藏项目"标签页

5. **再次访问**
   - 点击收藏列表中的项目卡片
   - 返回该项目的详情页

## 注意事项

- 重复收藏同一项目会提示"该项目已收藏"
- 收藏数据存储在本地，卸载小程序后会清空
- 项目列表按收藏时间倒序排列（最新收藏在前）
- 空状态会显示提示信息："暂无收藏的项目"
