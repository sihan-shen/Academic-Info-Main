# 导师社工小程序前端1

这是一个基于微信小程序开发的导师社工平台前端项目。

## 项目结构

- `pages/index/`: 首页，包含热点资讯、快捷入口。
- `pages/search/`: 查导师/搜索页，包含筛选器和导师列表。
- `pages/tutor-detail/`: 导师详情页，展示导师简介、论文、项目等信息。
- `pages/user/`: 个人中心页，展示用户信息、会员状态、功能菜单。
- `pages/favorites/`: 我的收藏页，包含收藏的导师、项目和关注风险。
- `pages/login/`: 登录页，支持账号密码登录。

## 功能特性

1. **首页**: 展示平台核心功能入口和最新资讯。
2. **导师查询**: 支持多维度筛选（学科、职称、院校等）和关键词搜索。
3. **导师详情**: 全方位展示导师学术成果、研究方向和项目经历。
4. **个人中心**: 管理个人信息、查看收藏、会员服务。
5. **收藏夹**: 统一管理感兴趣的导师和项目。

## 开发环境

- 微信开发者工具
- 基础库版本: 2.19.4 或更高

## 交互逻辑

- **登录流程**: 打开小程序 -> 首页 -> 点击"我的" -> 若未登录跳转登录页 -> 登录成功 -> 跳转首页/个人中心。
- **搜索流程**: 首页搜索框/查导师入口 -> 搜索页 -> 点击导师卡片 -> 导师详情页。
- **收藏流程**: 导师详情页 -> 点击收藏 -> 个人中心 -> 我的收藏 -> 查看收藏列表。

## 注意事项

- 图片资源目前使用占位符或下载的示例图片，实际开发中需替换为服务器资源。
- API 接口均为模拟数据，需对接后端服务。
