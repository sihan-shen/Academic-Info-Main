## 📊 字段映射详情

### 编辑页面 → 用户页面字段转换

```javascript
// 编辑页面保存的数据结构
editUserInfo = {
  avatar: '/temp/avatar.jpg',
  name: '田雯雯',
  phone: '138****5678',
  gender: '男',
  academicStatus: '硕士研究生',      // ← 注意这个字段名
  signature: '这位学者很懒...',       // ← 注意这个字段名
  institution: '清华大学',
  tutor: '张教授',
  achievement: '发表论文10篇'
}

// 用户页面显示的数据结构
userInfo = {
  avatar: editUserInfo.avatar,           // → '/temp/avatar.jpg'
  name: editUserInfo.name,               // → '田雯雯'
  phone: editUserInfo.phone,             // → '138****5678'
  role: editUserInfo.academicStatus,     // → '硕士研究生'
  bio: editUserInfo.signature,           // → '这位学者很懒...'
  vipStatus: editUserInfo.vipStatus,     // → false
  gender: editUserInfo.gender,           // → '男'
  institution: editUserInfo.institution, // → '清华大学'
  tutor: editUserInfo.tutor,            // → '张教授'
  achievement: editUserInfo.achievement  // → '发表论文10篇'
}
```

## 🐛 常见问题

### Q1: 修改后返回页面，数据没有更新？
**A:** 
- 检查是否成功保存（应该看到"XX已更新"提示）
- 检查控制台是否有错误信息
- 确认 `onShow()` 生命周期函数被调用

### Q2: 关闭小程序后数据丢失？
**A:**
- 检查本地存储是否成功（查看控制台日志）
- 确认没有手动清除缓存
- 检查本地存储容量是否超限（10MB）

### Q3: 某些字段不显示？
**A:**
- 检查字段映射是否正确（`academicStatus` → `role`）
- 确认 WXML 中使用了正确的变量名
- 查看数据是否成功加载（控制台日志）
